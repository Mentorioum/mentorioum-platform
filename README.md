# Mentorioum Platform

Mentorioum collaboration platform

## Requirements

1. `docker` - 19.03.8
2. `node` only starting from  14.x.x
3. `yarn` - 1.22.4

## Installation

Just run `yarn`

## Starting App

1. Run `docker-compose up`
2. Run `yarn start:dev`
